Welcome to the AGS Rest API!

The AGS is the Amtliche Gemeindeschlüssel AGS Search is a JSON Rest API to query the AGS by key or place (eg. city, village, county) for Germany.

Endpoints

/ags/ (query place by AGS-Key)

Example Requests
placerequest
Linz am Rhein/ags/07138041
München/ags/09162000
Bonn/ags/05314000


/place/ (query AGS-Key by place


 
Example Requests
placerequest
Linz am Rhein/place/Linz am Rhein
München/place/München
Bonn/place/Bonn
Info: Die Stadtstaaten Hamburg, Bremen und Berlin suchen sie via Stadt-/Ortsteile bzw. Stadtbezirke.
